# Contributing

## English

This project accepts practical improvements that make the stack easier to install, safer to operate, or clearer to understand.

Good contributions:

- safer relay setup
- clearer docs
- better macOS launchd support
- better panel checks
- backup improvements
- examples for Paper, Purpur, or Vanilla deployments

Do not include private deployment data, production logs, player data, world saves, plugin jars, or Minecraft server jars.

## 中文

欢迎提交能让这套方案更好安装、更安全、更容易理解的改进。

适合贡献的内容：

- 更安全的 relay 初始化
- 更清楚的文档
- 更好的 macOS launchd 支持
- 更准确的面板检测
- 更稳的备份功能
- Paper、Purpur、Vanilla 的部署示例

不要提交私有部署数据、生产日志、玩家数据、世界存档、插件 jar 或 Minecraft 服务端 jar。
